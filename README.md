# Token Presale + Vesting Smart Contract

A secure Solidity smart contract for crypto token presale with vesting and claim functionality.

## Features

- Token presale
- Vesting lock system
- Claim tokens after vesting
- ETH payment support
- Owner withdrawal
- OpenZeppelin security

## Tech Stack

- Solidity
- Hardhat
- OpenZeppelin
