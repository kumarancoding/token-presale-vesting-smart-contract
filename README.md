# Launch Token (LTT)

A Solidity ERC-20 token project built using Hardhat and deployed on the Ethereum Sepolia Testnet.

## Token Information

* **Token Name:** Launch Token
* **Token Symbol:** LTT
* **Network:** Ethereum Sepolia Testnet
* **Total Supply:** 1,000,000 LTT

## Features

* ERC-20 compliant token
* Fixed token supply
* Hardhat deployment scripts
* Sepolia testnet deployment
* Token information verification script

## Tech Stack

* Solidity
* Hardhat
* Ethers.js
* MetaMask
* Sepolia Testnet

## Installation

```bash
npm install
```

## Deploy Contract

```bash
npx hardhat run scripts/deploy.js --network sepolia
```

## Check Token Information

```bash
npx hardhat run scripts/check.js --network sepolia
```
